# 🐾 MyScreenPets

Welcome to **MyScreenPets**! This is a fun and interactive desktop companion application built with Electron. Choose your favorite animated pets to roam around your screen, keep you company while you work, and react to your mouse cursor!

## ✨ Features

- **Multiple Characters**: Select from different characters (like Pikachu, Gengar, Rayquaza, and more!).
- **Customizable Behaviors**: Adjust their scale, walking style (bounce, slide, hop), speed, and activity level.
- **Interactive**: Pets react to your cursor (stare, run away, or ignore).
- **Smooth Animations**: High-quality CSS and JS driven animations that bring sprites to life.
- **Unobtrusive**: Pets roam freely on a transparent click-through window without interrupting your workflow.

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/GreyPikachu/MyScreenPets.git
   ```
2. Navigate to the project directory:
   ```bash
   cd MyScreenPets
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start your desktop pet using:
```bash
npm start
```
*Note: You can also use `npx electron .` to run the app in your local environment.*

## ⚙️ Configuration

Right-click the pet to open the **Settings Menu**. Here you can:
- Switch the active character.
- Change the walking style and movement type.
- Adjust the scale of the character.
- Fine-tune activity levels and pause durations.

## 🛠 Built With

- [Electron](https://www.electronjs.org/) - Cross-platform desktop app framework.
- HTML, CSS, JavaScript - Classic web technologies for the frontend rendering.
